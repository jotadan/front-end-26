Introdução

C é uma linguagem de programação antiga e que serviu como base para a criação de outras linguagens. Estuda-se C para compreender conceitos simples de lógica e programação. Usa-se C em sistemas que fazem uso de pouco processamento e possuem pouca memória disponível.

Estrutura Básica

Bibliotecas

Se atente a iniciar os programas com a inclusão das bibliotecas desejadas. De inicio, vou passar apenas alguns comandos da stdio.h, stdlib.h, locale.h, stdbool.h e windows.h Nesse momento, lembre apenas que a stdio.h (Standard input Output) cuida das entradas, processamentos e saídas. Segue o exemplo: 

#include <nomedabiblioteca.h>
                #include <stdio.h>

Funções de cada uma delas

  • stdio.h: Entrada e saída de dados
  • stdlib.h: Utilidades gerais
  • stdbool.h: Tipo booleano
  • locale.h: Caracteres especiais

Main

A função main é o ponto de partida de todo programa em C. É por onde o código começa a ser executado. Geralmente é do tipo Int e retorna 0 Segue o exemplo



#include
int main(){
return 0;
}

Entrada e saída de dados

Em C, a biblioteca stdio.h é responsável pela maioria das operações de entrada e saída de dados. Existem algumas formas de coletar diferentes tipos de dados, algumas mais adaptáveis e seguras que as outras, contudo, iremos trabalhar apenas com duas.

printf

A função printf é responsável por apresentar os valores na tela. A sintaxe básica é:

int main(){
printf("Olá!");
return 0;
}
A função printf também pode ser usada para imprimir o valor das variáveis. Para isso, veremos os especificadores de formato.

Especificador Tipo de Dados
%c char
%s string
%d inteiro
%i inteiro
%f float
%lf double

scanf

A função scanf é usada para ler e armazenar um valor digitado pelo usuário. Ela recebe como parâmetros o tipo dado a ser armazenado e o endereço da variável para onde será armazenado.
scanf("%d", &idade);

Desvio Condicional

Conhecido também como Tomada de Decisão e Estrutura de Controle. Tem como objetivo desviar o caminho de execução dos comandos de acordo com a condição ou condições impostas juntamente com uso de operadores.

Simples

Este é o desvio condicional simples, é contido de uma ou mais condições e um único caminho, apenas caso a condição (ou condições) seja satisfeita. 

                
                    #include <stdio.h>
                    int main() {
                        float n1, n2, med; 
                    
                        printf("Primeiro nota: ");
                        scanf("%f", &n1);
                    
                        printf("Segundo nota: ");
                        scanf("%f", &n2);
                        med = (n1 + n2) / 2.0;
                    
                        if (med >= 6.0) {
                            printf("Media: %.2f\n", med);
                        }

                        return 0; 
                    }

Condicional

O Desvio Condicional composto é contido por uma ou mais condições e dois caminhos a seguir, ou seja, se verdadeiro toma-se o caminho IF, se falso, toma-se o caminho ELSE. 

                
                    #include <stdio.h>

                    int main() {
                        float n1, n2, med; 
                    
                        printf("Primeiro nota: ");
                        scanf("%f", &n1);
                    
                        printf("Segundo nota: ");
                        scanf("%f", &n2);
                    
                        med = (n1 + n2) / 2;
                    
                        printf("Media: %.2f\n", med);
                        
                        if (med >= 6.0) {
                            printf("APROVADO!\n");
                        } else {
                            printf("REPROVADO!\n");
                        }
                        
                        return 0; 
                    }

Encadeado/Aninhada

Tanto no desvio simples quanto no composto, podemos utilizar, se necessário for, desvio condicional encadeado, ou seja, a resultante de uma condição (else if) pode ser outra condição. 

                
                    #include <stdio.h>
                    
                    int main() {
                        float num; 
                    
                        printf("Informe um numero de 1 a 3: "); 
                        scanf("%f", &num); 
                    
                        if (num == 1) {
                            printf("Um\n");
                        } else if (num == 2) {
                            printf("Dois\n");
                        } else {
                            printf("Tres\n");
                        }
                        
                        return 0;
                    }

Múltipla Escolha

Este comando permite uma decisão múltipla, usamos quando formos comparar uma variável com vários valores possíveis. 

                
                    #include <stdio.h>
                    
                    int main() {
                        int dia; 
                    
                        printf("Digite um numero de 1 a 7: "); 
                    
                        scanf("%d", &dia); 
                    
                        switch (dia) {
                            case 1:
                                printf("Domingo\n");
                                break;

                            case 2:
                                printf("Segunda-Feira\n");
                                break;

                            case 3:
                                printf("Terça-Feira\n");
                                break;

                            case 4:
                                printf("Quarta-Feira\n");
                                break;

                            case 5:
                                printf("Quinta-Feira\n");
                                break;

                            case 6:
                                printf("Sexta-Feira\n");
                                break;

                            case 7:
                                printf("Sábado\n");
                                break;

                            default:
                                printf("Invalido, tente novamente!\n");
                        } 
                    
                        return 0; 
                    }
